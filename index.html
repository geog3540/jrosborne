Geovisualization Portfolio Joshie Osborne : GEOG 3540

Data Exploration

In this exercise, on Observable Notebook, I introduced myself with Observable Notebook, learning how to properly insert files and extract certain varibles from a .csv file. I also learned how and where to find appropriate geospatial data.

Data Classification

This exercise introduced me to data classification with JavaScript on Observable Notebook. I used the data from my data exploration exercise and calssified it with different classification schemes to find the most appropriate one. I tested the unclassed, quantile, natural breaks, equal interval, and manual threshold schemes.

US Population Density

This exercise involved using the data and classification scheme from the previous exercises to create a choropleth map. This introduced me to SVG and resulted in the creation of a choropleth map showing US population densities by county.

US Hydropower Plant Capacity

This exercise taught me how to correctly use point data to create a proportional symbols map displaying US hydropower plant capacities. In doing so, I learned how to manipulate and incorporate point data in Observable Notebook and ArcGIS Pro.

Population vs Vacancy in Florida

This exercise involved finding bivariate data and creating a bivariate map on Observable, both of which were new to me.
